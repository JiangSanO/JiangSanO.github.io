1 插入代码块

可以使用 <s 模板插入代码块模板,在后面选择插入的语言.

如果想要求值,将光标移到代码块内,按下快捷键 C-c C-c .

1.1 插入C语言代码块

# +begin_src C++ :includes <stdio.h>
  int a=1;
  int b=1;
  printf("%d\n", a+b);
# +end_src

# +RESULTS:
: 2

1.2 插入Python语言代码块

# +BEGIN_SRC python :results output
a = 1+1
print a
# +END_SRC

# +RESULTS:
: 2

2 使用Org的日程管理

2.1 打开Org日程管理

使用快捷键 C-c c 或 Spc X (doom emacs) 启动命令 org-capture .

文件的存放路径由变量 org-directory 决定. 

(setq org-directory "~/org/journal/")
