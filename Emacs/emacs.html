编译安装 emacs。

1.1 Linux (WSL)

安装依赖 

  sudo apt-get install --assume-yes build-essential libncurses5-dev
  sudo apt-get install --assume-yes libgif-dev libxpm-dev
  sudo apt-get install --assume-yes libtiff5-dev
  sudo apt-get install --assume-yes libxml2-dev
  sudo apt-get install --assume-yes libgnutls28-dev
  sudo apt-get install --assume-yes gtk+-3.0

然后进入如下命令： 

  ./autogen.sh;
  ./configure --prefix=/usr/emacs --with-x-toolkit=gtk;
  make;
  make install;

2 Emacs 基础

2.1 进入和退出 Emacs

  • C-x C-c save-buffers-kill-terminal 退出 emacs
  • C-z suspend-frame 在终端环境，暂停它；在 gui 环境，最小化

若存在未保存的缓冲区，emacs 会询问是否保存；当存在子进程时，emacs 也会询问是否关闭。

若 emacs 作为服务器使用，客户端按下 C-x C-c 只会关闭客户端进程。

emacs 能够方便的记录某一个会话信息在你关闭它时。

你可以配置变量 confirm-kill-emacs 来设置退出时的操作，变量接受一个谓 词函数。若谓词函数返回 nil ，那么 emacs 会继续进行，否则退出。
常见的谓词函数是: yes-or-no-p (一般谓词函数都以 p 结尾)

若变量 confirm-kill-processes 的值是 nil ，那么在关闭 emacs 不会询问任何 是否关闭子进程的信息，若为 t，则默认询问。

更加高功能的自定义，请看 Section “Killing Emacs” in The GNU Emacs Lisp Reference Manual.

使用命令 kill-emacs 能够不在任何提示下关闭 emacs。

2.2 基本编辑

2.2.1 字符插入

使用命令 (newline) 即可插入新行。新行会根据当前的主要模式进行缩进。

如果不想任何的自动缩进，可以按下 C-j ，它仅仅插入一个新行。

你可以使用次要模式改变 emacs 的文本插入操作，例如自动换行模式 auto-fill-mode 或者改写模式 overwrite-mode.

当你想要插入其他的非图形字符，可以使用命令 C-q 加上对应的按键，比如 C-q DEL 将插入字符 DEL, 在 emacs 以符号 ^? 表示。
也可以使用任意多的八进制序列来插入字符，以非八进制数字作为结束，比如 C-q 1 0 1 B 将插入 AB。
不过对于非二进制的改写模式，输入八进制序列仅仅在光标处插入数字而已。

要想改变输入的进制序列，可以设置变量 read-quoted-char-radix 为 10 或 16. 分别表示 10 进制和 16 进制。
当它是 16 进制时，那么字母 a~f 也作为数字表示，并且忽略大小写。

一些常用的 Unicode 字符可以使用 C-x 8 来插入，比如 C-x [ 插入全角左 引号， C-x 8 ] 插入全角右单引号， C-x 8 { 插入全角左双引号， C-x 8 } 则插入全角右双引号。
也可以使用 Alt 按键来快速输入，比如 A-[ 等价于 C-x 8 [, 要查看关 于 C-x 8 的所有短的快捷键，输入 C-x 8 C-h.

你也可以使用 C-x 8 RET (insert-char) 来通过 Unicode 代码点或 Unicode 字符名来输入,如果你键入名字， 该命令会为你进行补全，如果你键入代码点， 要么使用十六进制（Unicode 的约定），或者使用指定进制的 elisp 合法格式，比 如 #o23072 表示八进制。 see Section “Integer Basics” in The Emacs Lisp Reference Manual.

可以使用前缀参数 C-u 到命令 C-q 或 C-x 8, 来指定需要插入多少个字 符。不同命令的前缀参数完全不一样。

2.2.2 光标移动

M-r (move-to-window-line-top-bottom) 循环将光标移到屏幕中间，屏幕 顶部，屏幕底部。
若使用前缀参数，那么将移到指定“屏幕行”，屏幕行以 0 开始（表示顶部）， 负数表示从底部计数，从-1 开始。

M-g c 读入一个数字，移动到指定的字符位置。 (goto-char)

M-g M-g M-g g 读入一个数字，移动到指定行。可以指定一个前缀参数，获 得移动的次数 (goto-line)

M-g TAB 读入一个数字，移动到指定列。可以指定一个前缀参数，获得移动的 次数。

C-x C-n 设置临时的全局列位 (the semipermanent goal column) ，设置 后，使用 C-p 或 C-n 移动时总是设法移动到该列。 (set-goal-column)

C-u C-x C-n 取消设置全局列位。

2.2.3 文本删除 4.3 41L

在 emacs 中，向前（forward）表示有行尾方向走去，而向后（backward）表示往 行首方向走去。

在 emacs 文档中，DEL 键等价于按键 backspacs 而不是 Delete 键。

在行尾使用 C-k (kill-line) 能够加入后一行到当前行。

2.2.4 撤消编辑 4.4 42L

undo 命令只能撤消文本的改变，无法还原光标位置的变化，不过当你还原时， 光标会出现在改变前的位置。

2.2.5 帮助 4.6 43L

C-h k 表示查找快捷键的帮助。

2.2.6 空白行 4.7 43L

C-o 插入一个空白行在当前光标之后，但不一移动光标。

C-x C-o 删除所有的连续空行，但不包括其中一条，若光标位于非空行，那么 将删除光标之下的所有空行。

2.2.7 光标位置信息 4.9 44L

  • 当前光标位于哪一行
  • 在状态栏动态显示行号
  • 显示文本域的行数，单词数，字符数
  • 显示缓冲区的行数，单词数，字符数
  • 显示光标当前的字符信息（字符编码，位于缓冲区的位置等）
  • 突出当前行
  • 动态显示当前缓冲区大小

(defun my--set-font (&optional frame)
  (with-selected-frame (or frame (selected-frame))
    (if (string-equal system-type "windows-nt")
        ;; 下面是用于Windows的配置。
        (progn
          ;; 设置英文字体为Inconsolata，并指定字号18。
          ;; 因为不同操作系统下字体显示的大小不一样(DPI的问题)，所以分开设置。
          (set-face-attribute 'default nil :font "mononoki Nerd Font 12")
          ;; 给相应的字符集设置中文字体，这里的字体是冬青黑体简体中文 W3。
          (dolist (charset '(han cjk-misc chinese-gbk))

                              charset (font-spec :family "Microsoft YaHei UI")))
          ;; 设置默认字体和用于显示特殊符号的字体，如果不需要下面三句可以删除。
          (set-fontset-font "fontset-default"

2.6.11 org-download-clipboard

;;; org-download from clipborad
(defun org-download-clipboard (&optional captionp)
  "Save the captured image from clipboard to file, and insert into buffer. Or org-download-yank.
1. 使用任何应用软件截屏或者复制图片到剪贴板，在org-mode里粘贴即可。图片会自动保存到当前文件目录下的images文件夹里。
2. 该函数支持粘贴剪贴板的文字和图片，所以你可以将Ctrl-y绑定到该函数上，然后如粘贴文字一样粘贴图片，当然文字也OK。
3. Ctrl-u Ctrl-y会提示为图片命名，并在文档中引用该图片，同时会提示为图片插入caption。
4. 在org capture buffer中粘贴时，该函数会找到目标文件的目录，并将图片保存到该目录中。"
  (interactive "P")
  (let* ((image-dir
          (if (not (buffer-file-name))
              (cond ((string-prefix-p "CAPTURE-[0-9]" (buffer-name))
                     (let ((buffer-name (replace-regexp-in-string "CAPTURE-[0-9-]*" "" (buffer-name))))
                       (concat (file-name-directory (buffer-file-name (get-file-buffer buffer-name))) "./Material/Image")))
                    (t (yank) (error "")))
            "./Material/Image"))
         (fname (concat (make-temp-name "image-") (format-time-string "%Y%m%d-%H%M%S")))
         (image-file (concat image-dir "/" fname ".png"))
         (exit-status
          (call-process "convert" nil nil nil
                        "clipboard:" image-file)))
    (if (zerop exit-status)
        (progn
          (unless (file-exists-p image-dir) (make-directory image-dir))
          (if captionp
              (let ((rename (read-string "Filename to rename the temp images: ")))
                (rename-file image-file (concat image-dir "/" rename ".png") t)
                (insert (format "#+CAPTION: %s label:fig:%s\n" (read-string "Caption: ") rename))
                (kill-new (format "Fig. ref:fig:%s " rename)))
            (insert (format "[[file:%s]]" image-file))
            (org-display-inline-images)))
      (when captionp (user-error "No images in clipboard."))
      (yank))))

3 spacemacs 使用指南

3.1 文本操作

字符计数 space x c 对文本块内的字符进行计数,他会区分中文字符和英文字符.

3.2 常见快捷键

3.2.1 配置文件

SPC f e d 快速打开配置文件

SPC f e R 同步配置文件

SPC q q 退出 Emacs

SPC q R 重启 Emacs

3.2.2 文件管理

SPC f f 打开文件

SPC f t neotree 方式显示文件路径

Ctrl s 搜索当前文件（需安装 ivy layer）

  • 另一种搜索文件的姿势（需将光标置于需搜索的单词处）
    • n 下一个匹配
    • N/p 前一个匹配
    • r 改变范围：当前屏幕，当前函数，当前 buffer
    • e 编辑所有匹配（类似于替换）
    • / 在当前 project 搜索
    • SPC s c 清除搜索高亮

SPC f R 重命名当前文件

SPC f E 使用 sudo 来编辑文件（当某些文件的权限是只读的时候）

SPC f D 删除当前文件

SPC f r 打开最近文件列表（需安装 ivy layer）

SPC f y 复制当前文件的绝对路径

SPC f c 复制文件

3.2.3 buffer 管理

SPC b b 显示已经打开的 buffer

SPC b d 关闭当前 buffer

SPC b h 进入 Spacemacs 初始界面

SPC b N 新建一个 buffer

SPC b R 从自动备份的文件中恢复

SPC b Y 复制整个 buffer 的内容

SPC b P 将剪贴板的内容粘贴到整个 buffer

SPC Tab 切换至上一个 buffer

3.2.4 窗口管理

SPC n(number) 跳转至第 n 号窗口

SPC 0 跳转至 neotree 侧边栏

SPC w m 当前窗口最大化

SPC w s 或 SPC w - 水平分割窗口

SPC w v 或 SPC w / 竖直分割窗口

SPC w = 平衡窗口

SPC w d 删除当前窗口

SPC w o 切换至其他窗口

SPC t g 将当前窗口与其他窗口 黄金分割

3.2.5 project 管理

SPC p f 在当前 project 中查找文件

SPC p p 切换项目

SPC / 在该项目中搜索字符串

SPC p R 在项目中替换字符串，先输入「匹配」的，再输入「替换」的字符串（我一般不使 用这种方式，我用*来替换）

3.2.6 缩进代码

SPC j = 自动对齐

3.2.7 shell 操作

SPC ’ 打开/关闭 Eshell（需安装 shell layer）

SPC a s 打开其它种类的 Shell

3.2.8 中断操作

C g 输错命令时，可取消该次输入

3.2.9 显示动态行号

将 dotspacemacs-line-numbers 的值改为 ‘relative
